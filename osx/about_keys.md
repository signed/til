# About keys on macOS
- ^ is the ctrl key
- ⌘ is the command key
- ⌥ is the option/alt key
- ⇧ is the shift key

## Keyboard ##

```
    ~   ALT + n
    @   ALT + l  followed by space
    \   ALT + SHIFT + /
    |   ALT + 7
    [   ALT + 5
    ]   ALT + 6
```

## Finder

```
shift + cmd + g open dialog to enter a path
```

## Short Cuts ##

```
alt + cmd  <- ->
cmd + ,               open settings
cmd -                 comment
cmd + ß               fold
cmd + ´               unfold
cmd + shift +  <- ->  Resize windows
```
